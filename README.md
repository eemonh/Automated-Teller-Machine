# Automated-Teller-Machine
A robust, secure, and scalable backend engine designed to simulate core banking operations for an Automated Teller Machine.

Core Functionalities:
1. Secure Authentication
2. Transaction Processing
3. Security & Auditing

Can modify more complex funtionality. such as, Real-time balance inquiries and automated updates to the ledger upon successful transactions.Monitoring of the physical cash available within the ATM unit to trigger replenishment alerts. Based upon the system and machine I can update the engine that would be necessary. like making ledger, connecting DB and so on.
